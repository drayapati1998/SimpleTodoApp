# SimpleTodoApp
This is a simpleTodo app built using both frontend and backend with React,CSS,Node.js & Express and sqlite as Database
This app allows users to add new todos,view todos and edit todos and delete todos
I have 2 folders inside simpleTodo app one is client and other is server
How to run a project:
1.start Backend with
->cd server
->npm install
->npm install nodemon
->node index.js(server will run on 5000)
2.start Frontend with
->cd client
->npm install
->npm start(Frontend runs on 3000)
